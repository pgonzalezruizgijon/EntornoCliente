# [Tic Tac Toe](https://youtu.be/LD9UEC7AI7g)
- Coding Tutorial: [https://youtu.be/AzvpHNkjqsg](https://youtu.be/LD9UEC7AI7g)
- Demo: https://imkennyyip.github.io/tictactoe/

In this tutorial, you will learn how to make the simple game of tic tac toe. You will learn how to use css to style and organize the game in a 3x3 grid structure, and javascript to create and update web elements on the page while making this game.
